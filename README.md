# AUZA v2.1 — PDF-to-HWP 스마트 작성기

PDF에서 텍스트/표/이미지를 캡처하고 AI(Gemini)로 변환하여 한글(HWP) 문서를 자동 생성하는 Windows 데스크톱 앱입니다.

## 시스템 요구사항

| 항목 | 요구사항 |
|------|----------|
| OS | Windows 10/11 (64-bit) |
| 한글 | 한컴오피스 한글 2020 이상 (HWP 내보내기용) |
| Gemini API 키 | [Google AI Studio](https://aistudio.google.com/apikey)에서 무료 발급 |

> 한글 프로그램이 없어도 PDF 캡처 → 에디터 편집까지는 사용 가능합니다. HWP 내보내기 시에만 한글이 필요합니다.

## 설치

### 설치 버전
`AUZA-v2 Setup 2.1.0.exe` 실행 → 자동 설치

### 무설치 버전 (포터블)
`AUZA-v2-2.1.0-portable.exe` 바로 실행 (설치 불필요)

## Gemini API 키 설정

1. [Google AI Studio](https://aistudio.google.com/apikey)에서 API 키를 발급받습니다
2. 앱 첫 실행 시 자동으로 설정 모달이 열립니다
3. API 키를 입력하고 **저장**을 클릭합니다
4. 이후 앱을 재시작해도 키가 유지됩니다

> API 키는 사용자 PC의 `%APPDATA%/AUZA-v2/config.json`에만 저장되며, exe 파일에 포함되지 않습니다.

## 사용법

### 기본 워크플로우

```
PDF 열기 → 영역 캡처 → AI 변환 → 에디터 편집 → HWP 내보내기
```

### 1. PDF 열기
상단 **PDF 열기** 버튼을 클릭하여 PDF 파일을 엽니다.

### 2. 영역 캡처
**캡처** 탭을 선택한 뒤 PDF 위에서 드래그하여 변환할 영역을 선택합니다.

### 3. 에디터 편집
캡처된 내용이 오른쪽 에디터에 자동 삽입됩니다. 툴바를 사용하여 수정할 수 있습니다.

### 4. HWP 내보내기
한글 프로그램을 열고 커서를 원하는 위치에 놓은 뒤, 블록의 **HWP** 버튼을 클릭합니다.

## 캡처 모드

| 모드 | 설명 |
|------|------|
| **일반** | 선택 영역을 Gemini Vision으로 직접 인식 |
| **OD ON** | AI가 레이아웃(텍스트/표/이미지/수식)을 자동 감지하여 영역별 최적 변환 |
| **Review ON** | OD 감지 결과를 사용자가 편집(삭제/타입 변경/리사이즈) 후 변환 |
| **IMG** | 이미지 크롭 전용 모드 (고해상도) |

## OD Review 모드

Review ON 상태에서 캡처하면 감지된 영역을 편집할 수 있습니다:

- **클릭**: 영역 선택
- **드래그**: 영역 이동
- **코너/엣지 핸들**: 8방향 리사이즈
- **Delete**: 선택 영역 삭제
- **드롭다운**: 영역 타입 변경 (text/table/figure/formula/abandon)
- **+ 영역 추가**: 새 영역 수동 그리기
- **Ctrl+C/V**: 영역 복사/붙여넣기
- **Enter**: 변환 실행
- **Esc**: 취소

## 단축키

| 단축키 | 기능 |
|--------|------|
| 마우스 드래그 (캡처 모드) | 영역 캡처 |
| Delete (OD Review) | 선택 영역 삭제 |
| Enter (OD Review) | 변환 실행 |
| Esc (OD Review) | 리뷰 취소 |
| Ctrl+C/V (OD Review) | 영역 복사/붙여넣기 |

## 알려진 제한사항

- HWP 내보내기는 **한글 프로그램이 실행 중**이어야 합니다
- 수식 변환은 LaTeX 기반이며, 복잡한 수식은 이미지 fallback으로 처리됩니다
- OD 모델 첫 실행 시 모델 다운로드에 시간이 소요될 수 있습니다
- figure 이미지 해상도는 캡처 영역 크기에 비례합니다 (큰 영역 = 높은 해상도)
